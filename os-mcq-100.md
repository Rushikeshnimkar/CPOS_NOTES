## Section 1: Introduction to Operating Systems (Q1-15)

**Q1. [Easy]** An Operating System acts as an intermediary between:
- a) Hardware and user
- b) Compiler and interpreter
- c) Hardware and peripheral devices
- d) User and database

**Answer: a) Hardware and user**

---

**Q2. [Easy]** Which of the following is the core component of an Operating System that is always running?
- a) Shell
- b) Kernel
- c) Compiler
- d) GUI

**Answer: b) Kernel**

---

**Q3. [Easy]** Which type of operating system is designed for applications that require guaranteed timing?
- a) Batch OS
- b) Distributed OS
- c) Real-Time OS
- d) Network OS

**Answer: c) Real-Time OS**

---

**Q4. [Easy]** In a Time-Sharing Operating System, the fixed time period allocated to each process is called:
- a) Time slice/Quantum
- b) Context time
- c) Response time
- d) Turnaround time

**Answer: a) Time slice/Quantum**

---

**Q5. [Easy]** Which is NOT a function of an Operating System?
- a) Process Management
- b) Memory Management
- c) Compiling programs
- d) File Management

**Answer: c) Compiling programs**

---

**Q6. [Medium]** Which operating system architecture has all OS services running in kernel space in a single large block?
- a) Microkernel
- b) Monolithic
- c) Layered
- d) Hybrid

**Answer: b) Monolithic**

---

**Q7. [Medium]** What is the main advantage of Microkernel architecture?
- a) Fast performance
- b) Easy to implement
- c) More reliable (service crash doesn't crash system)
- d) Less memory usage

**Answer: c) More reliable (service crash doesn't crash system)**

---

**Q8. [Medium]** A system call is:
- a) A function call made by the operating system to hardware
- b) A programmatic way for a program to request a service from the kernel
- c) An interrupt generated by hardware
- d) A method to clone a process

**Answer: b) A programmatic way for a program to request a service from the kernel**

---

**Q9. [Medium]** Which is an example of a Distributed Operating System advantage?
- a) Simple to implement
- b) No network dependency
- c) Fault tolerance
- d) Single point of failure

**Answer: c) Fault tolerance**

---

**Q10. [Medium]** In Hard Real-Time OS, if a deadline is missed:
- a) System continues normally
- b) Task is retried
- c) System failure occurs
- d) Task is queued

**Answer: c) System failure occurs**

---

**Q11. [Hard]** Which system call is used to create a new process in UNIX?
- a) exec()
- b) fork()
- c) create()
- d) spawn()

**Answer: b) fork()**

---

**Q12. [Hard]** The mode bit in OS is used to distinguish between:
- a) User mode and kernel mode
- b) Read and write mode
- c) Active and inactive mode
- d) Online and offline mode

**Answer: a) User mode and kernel mode**

---

**Q13. [Hard]** Which of the following is NOT a type of system call?
- a) Process control
- b) File management
- c) Memory compilation
- d) Device management

**Answer: c) Memory compilation**

---

**Q14. [Hard]** A trap is a:
- a) Hardware-generated interrupt
- b) Software-generated interrupt
- c) Network interrupt
- d) Timer interrupt

**Answer: b) Software-generated interrupt**

---

**Q15. [Hard]** Which architecture does Windows NT use?
- a) Monolithic
- b) Microkernel
- c) Hybrid
- d) Layered

**Answer: c) Hybrid**

## Section 4: Processes & Scheduling (Q46-65)

**Q46. [Easy]** A process is:

**Q47. [Easy]** Which memory section contains the program code?
- a) Stack
- b) Heap

**Answer: c) Text**

---

**Q48. [Easy]** When a process is waiting for I/O, it is in which state?
- a) Ready
- b) Running
- c) Blocked/Waiting

**Answer: c) Blocked/Waiting**

---

**Q49. [Easy]** The PCB (Process Control Block) is also known as:
- a) Task Control Block
- b) Memory Control Block
- c) CPU Control Block
- d) I/O Control Block

**Answer: a) Task Control Block**

---

**Q50. [Easy]** Which scheduling algorithm serves processes in the order they arrive?
- a) SJF
- b) FCFS
- c) Round Robin
- d) Priority

**Answer: b) FCFS**

---

**Q51. [Medium]** If a parent process terminates before its child, the child becomes a(n):
- a) Zombie process
- b) Orphan process
- c) Daemon process
- d) Dead process

**Answer: b) Orphan process**

---

**Q52. [Medium]** Which scheduling algorithm may cause the Convoy Effect?
- a) SJF
- b) FCFS
- c) Round Robin
- d) Priority

**Answer: b) FCFS**

---

**Q53. [Medium]** In Round Robin scheduling with time quantum 4, if a process has burst time 10, how many times will it be scheduled?
- a) 2
- b) 3
- c) 4
- d) 10

**Answer: b) 3** (4+4+2 = 10)

---

**Q54. [Medium]** A zombie process is:
- a) A process waiting for I/O
- b) A terminated child whose parent hasn't called wait()
- c) A process in infinite loop
- d) A process with no memory

**Answer: b) A terminated child whose parent hasn't called wait()**

---

**Q55. [Medium]** Which scheduler is responsible for swapping processes between main memory and disk?
- a) Long-term scheduler
- b) Short-term scheduler
- c) Medium-term scheduler
- d) CPU scheduler

**Answer: c) Medium-term scheduler**

---

**Q56. [Hard]** Given processes with arrival time 0 and burst times: P1=24, P2=3, P3=3. Using FCFS, what is the average waiting time?
- a) 17
- b) 27
- c) 10
- d) 30

**Answer: a) 17**
> WT(P1)=0, WT(P2)=24, WT(P3)=27. Average = (0+24+27)/3 = 17

---

**Q57. [Hard]** Which scheduling algorithm is known to have the minimum average waiting time?
- a) FCFS
- b) SJF (Shortest Job First)
- c) Round Robin
- d) Priority

**Answer: b) SJF (Shortest Job First)**

---

**Q58. [Hard]** SRTF (Shortest Remaining Time First) is the preemptive version of:
- a) FCFS
- b) Priority Scheduling

**Answer: c) SJF**

**Q60. [Hard]** Aging is a solution to which problem in Priority Scheduling?

**Q62. [Hard]** In a semaphore, if the count is -3, how many processes are waiting?

**Q64. [Hard]** Which IPC mechanism provides the fastest communication?

**Q65. [Hard]** Waiting Time is calculated as:

## Section 5: Memory Management (Q66-80)

**Q66. [Easy]** In paging, physical memory is divided into fixed-size blocks called:
- a) Pages
- b) Frames
- c) Segments
- d) Blocks

**Answer: b) Frames**

---

**Q67. [Easy]** Logical memory is divided into fixed-size blocks called:
- a) Frames
- b) Pages
- c) Segments
- d) Partitions

**Answer: b) Pages**

---

**Q68. [Easy]** Which type of fragmentation occurs in paging?
- a) External fragmentation
- b) Internal fragmentation
- c) Both
- d) None

**Answer: b) Internal fragmentation**

---

**Q69. [Easy]** Page table maps:
- a) Physical address to logical address
- b) Page number to frame number
- c) Frame number to page number
- d) Segment to page

**Answer: b) Page number to frame number**

---

**Q70. [Medium]** If page size is 4 KB and logical address is 8196, what is the page number?
- a) 0
- b) 1
- c) 2
- d) 3

**Answer: c) 2**
> Page Number = 8196 / 4096 = 2

---

**Q71. [Medium]** Which memory allocation strategy selects the smallest hole that fits?
- a) First Fit
- b) Best Fit
- c) Worst Fit
- d) Next Fit

**Answer: b) Best Fit**

---

**Q72. [Medium]** The page table entry's Valid/Invalid bit indicates:
- a) Page has been modified
- b) Page is in memory or not
- c) Page is read-only
- d) Page has been accessed

**Answer: b) Page is in memory or not**

---

**Q73. [Medium]** External fragmentation can be solved by:
- a) First Fit
- b) Best Fit
- c) Compaction
- d) Worst Fit

**Answer: c) Compaction**

---

**Q74. [Medium]** In variable partitioning, which strategy allocates the largest available hole?
- a) First Fit
- b) Best Fit
- c) Worst Fit
- d) Next Fit

**Answer: c) Worst Fit**

---

**Q75. [Hard]** A system has 32-bit logical address and 8 KB page size. How many bits are needed for the page offset?
- a) 10 bits
- b) 12 bits
- c) 13 bits
- d) 14 bits

**Answer: c) 13 bits**
> 8 KB = 2^13, so 13 bits for offset

---

**Q76. [Hard]** TLB hit ratio is 98%, TLB access time is 20ns, memory access time is 200ns. What is EMAT?
- a) 220 ns
- b) 224 ns
- c) 240 ns
- d) 420 ns

**Answer: b) 224 ns**
> EMAT = 0.98×(20+200) + 0.02×(20+400) = 215.6 + 8.4 = 224 ns

---

**Q77. [Hard]** In segmentation, what happens if offset >= limit?
- a) Page fault
- b) Segmentation fault
- c) Memory overflow
- d) Translation continues

**Answer: b) Segmentation fault**

---

**Q78. [Hard]** For a 4 GB virtual address space with 4 KB pages and 4-byte page table entries, what is the page table size?
- a) 1 MB
- b) 2 MB
- c) 4 MB
- d) 8 MB

**Answer: c) 4 MB**
> Pages = 2^32 / 2^12 = 2^20. Size = 2^20 × 4 = 4 MB

---

**Q79. [Hard]** Internal fragmentation in paging is at most:
- a) Page size
- b) Page size - 1
- c) 2 × Page size
- d) Half of page size

**Answer: b) Page size - 1**

---

**Q80. [Hard]** Two-level paging is used to:
- a) Increase page table size
- b) Reduce page table memory overhead
- c) Eliminate paging
- d) Combine with segmentation

**Answer: b) Reduce page table memory overhead**
